# GitHub Learning Garden

A tiny, dependency-free website we will use to learn Git and GitHub safely.

## Open it locally

Double-click `index.html`, or open it in a browser. Click **Plant a seed** to test the small interaction.

## What we will practise

1. Git repository and the working folder
2. Commit and push
3. Branches
4. Pull requests and review
5. Merge versus squash merge
